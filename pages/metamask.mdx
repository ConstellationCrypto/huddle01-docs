# Add Huddle01 to Metamask
## Using the Huddle01 Hub Page
1. Navigate to our [Hub Page](https://huddle01.hub.caldera.xyz) 
2. Click the 
**Connect** button in the *Your Wallet* section, and you will be prompted to switch to the Huddle01 Network

![hubpage](../images/hub.png)

## Manually via Metamask
You can add the network manually via the following details:
- Network Name: Huddle01
- RPC URL: https://huddle01.calderachain.xyz/http
- Chain ID: 12323
- Currency Symbol: ETH
